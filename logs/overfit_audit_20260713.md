# 2026-07-13 过拟合/未来函数审计

## 你的问题
> 这个参数的胜率是怎么做到那么高的? 有没有过拟合或者未来函数?

## 结论一句话
**代码无未来函数, 但有"风格匹配+轻微过拟合"嫌疑**。Walk-forward 8 子窗口验证: 6/8 窗口盈利, 收益分布**严重偏向 2025-12 之后**。

## 5 个怀疑点的逐项排查

### 1. 未来函数 — 无 ✅
- `get_value_on(chart_pts, target_date)`: 严格 `d <= target_date` 截断
- `compute_score_at(code, cutoff_date, ...)`: 评分用 `p[0] <= cutoff_date` 截断
- 大佬交易: 用 `_date_prefix` (实际成交日) 而非 API 返回时间
- 基准 110020: 同样按日期截断

### 2. 业绩归因 — **5 维评分是动量因子** ⚠️
V2 的 quality 评分核心:
```python
ret_1y = valid[-1][1] - valid[-250][1]   # 过去 1 年收益
q_ret = min(2.5, max(0, 1.25 + ret_1y * 5))   # 1y +20% 收益 -> 2.5
```
**这是趋势跟随策略**。在 2024-2026 中国基金牛市里:
- 选中的都是"过去 1 年涨得猛"的基
- 涨得猛的基继续涨
- 表现自然好

**但这有个致命问题**: 在 2022-2023 熊市, 动量因子会失效甚至反向, 这套策略会崩。
- 当前回测区间 2024-03 ~ 2026-07 = 28 个月, 全是牛市区间
- **没有熊市样本, 没人敢保证它在熊市也能赚 122%**

### 3. 网格搜索过拟合 — 中等 ⚠️
- 第一轮网格 (TP×TRAIL, 12 组) 用的 2025 全年
- 第二轮扩展网格 (HOLD×max_hold×min_buy, 24 组) 用的 2024-03~2026-07 全段
- **没有"训练集 vs 验证集"严格分离** — 24 组参数都在 28 个月同段评估
- 这意味着**冠军是"在 28 个月上找的最优",换段可能不是最优**

### 4. 胜率计算 — 真实但有偏差 ⚠️
```python
win_sells = [s for s in sells if s.get("pnl_pct", 0) > 0]
win_rate = len(win_sells) / len(sells) * 100
```
- 卖出原因 `consensus_sell` 的 pnl_pct **写死为 0** (line 389)
- 6 笔 consensus_sell 不算 win 也不算 loss (从分母里算但 numerator=0)
- 这没大问题, 但**胜率 63.1% 是"非共识卖出原因中, pnl>0 的比例"**

## 关键洞察
1. **CHAMPION 平均收益比 BASELINE 高 67%** (12.49% vs 7.47%) — 真改善
2. **CHAMPION 胜率 75% vs BASELINE 88%** — 冠军**没那么稳**, 牛市赚得多熊市亏得多
3. **CHAMPION 收益分布严重右偏**: 2025-12 之后占大头
4. **CHAMPION 在 2025-09~11 也亏 6%** — 不是只赚不赔的圣杯
5. **平均回撤 5.12% 比 BASELINE 3.67% 大** — 高收益换高波动

## 胜率 63.1% 的本质解释
**不是"我找到一个能 60% 胜率的策略"**, 而是:
- **TP=15% 止盈在 2024-2026 牛市触发频繁** (38 笔, 平均 +17.24%) — 这是高胜率主因
- **time_tp (30 天未涨就退)** 触发 48 笔, 平均 -0.69% — 这是"试错"成本
- **市场状态 + 动量选基 + 严格止盈 + 短持仓** 配合完美, 牛市里收益爆炸

**但熊市会怎样?** 不知道, 没数据。**这是最大的不确定性**。

## 给你的建议

   `bear_market_no_buy: True` (熊市不买, 等牛市再来)

4. **如果 2026 Q3 出现熊市信号** (沪深 300 跌破 MA60), 立即切回 BASELINE (tp=20/8/60/3) — 回撤小
