## 💡 V2 比 V1 低的真相

### V1 "假象" 的本质

V1 回测引擎只有一种卖出方式：**止损 -10%**。盈利的基金永不主动止盈。

```
V1 收益来源 = 几只在回测期内翻倍的基 × 全部资金集中 × 永不止盈
V1 年化 53% = "拿死" 几只翻倍基 + 跟买加持仓
```

V1 的"低回撤 -9.38%"也是假象 — 实际是少数基的浮盈掩盖了多数被止损的亏损。

### V2 "真本事" 的本质

V2 加了 3 种止盈方式：
- **固定止盈 +15%**：13 笔，平均 +17.55%（主力收益）
- **移动止盈 -8%**：16 笔，平均 +3.51%（防坐电梯）
- **时间止盈 60d**：10 笔，平均 -0.07%（及时止错）

```
V2 收益来源 = 主动收割 60 笔出场 (13 笔吃+17% + 16 笔防回吐 + 31 笔微利/止损)
V2 年化 39% = 真正穿越牛熊的策略收益
```

## 🔬 P3 改造为何失败

P3 试图用 5 维评分当"持仓警告" — 评分低的基用更紧止盈。结果：
- 年化 39% → **17%** (-22%)
- 胜率 55% → 60.9% (胜率提升但收益降低)
- 交易笔数 60 → 87 (过度交易)
- **take_profit** 13 笔 → **44 笔** (阈值降低，过早止盈)

**根因** — 大佬共识买入的"低分基"反而是后续暴涨的黑马（机器评分短期预测力弱）。5 维评分把"潜力黑马"过早收割了。
