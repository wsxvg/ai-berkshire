## 三、参数优化的经验教训

### 教训1：贪心搜索不够全面

单参数逐级搜索可能错过**参数组合之间的交互效应**。比如"低买入门槛 + 低止损"可能比"低门槛 + 高止损"好，但贪心搜索不会测试这种组合。

### 教训2：最低评分 2.5 是过拟合

`min_score=2.5` 在这个 6 个月窗口里表现好，但不一定能推广。因为降低门槛让更多基金进入候选，正好这半年牛市，买什么都涨。换成震荡市可能就亏了。**3.3 作为默认值更安全。**

### 教训3：半年的数据量太小

6 个月、137 个交易日、273 只基金有足够数据——但市场环境只有**一个**（2026 上半年是 AI/半导体/商品牛市）。没有经历下跌市，无法验证风控能力。

### 教训4：权重差异在此市场下无意义

所有权重组合结果相同，因为在这个牛市中，高分基金的五个维度都高分，没有出现"质量好但动量差"的取舍场景。**权重可能需要跨周期才能调出差异。**

## 四、最终推荐参数

```python
# 保守版（推荐实际使用）
min_score = 3.3        # 避免买垃圾基金
take_profit_pct = 30   # 赚30%卖一半

cash_reserve = 0.20    # 留20%现金
max_position = 25      # 单只不超25%
kelly_cap = 0.20       # 凯利上限20%
weights = 25/20/20/15/20  # 默认权重
```

这些参数在回测中的保守表现：
- 一次性：+44.56%，回撤 6.51%
- 月薪定投：+21.20%，回撤 1.37%

## 六、总结

**回测证明了什么：**
- 五维评分 + 四级买卖信号 + 凯利分配 = 半年跑赢大盘 **3~7 倍**
- 月薪定投 + 评分引擎 = **+31% 收益，回撤仅 5.8%**
- 最差情况下（纯规则、无 AI 辅助）也能大幅跑赢基准

**回测没证明的：**
- 没有经历下跌市，不知道风控够不够硬
- 6 个月太短，长期（3 年+）表现未知
- 权重调优需要跨周期数据

**下一步建议：**
1. 管道持续跑，积累到 12 个月以上数据再做一次回测
2. 把量化规则抽成 `tools/fund_rules.py`，SKILL 减重
