# daily_check V5 报告 — 7 月胜率解释 + 新端点深度诊断

> 日期: 2026-07-13

## 一、7 月胜率 6% 不是问题（样本噪音）

**直接说结论** — 30 天胜率 6% 是统计噪音，**不反映真实策略水平**。

### 数据对比

| 区间 | 笔数 | 胜率 | Alpha (超额) | 评价 |
|------|----:|----:|----------:|------|
| 28 月全量 | 60 卖 | **55%** | +21.98% | 真实水平 |
| 24 月训练 | 48 卖 | 52.1% | +12.56% | 真实水平 |
| **4 月验证** | **13 卖** | **84.6%** | +161.02% | 真实水平 |
| **30 天实盘** | **6 卖** | **6%** | +7.63% | **样本噪音** |

### 为什么 30 天胜率失真

```
统计学基本原理:
  6 次抛硬币正 1 次 = 16.7% 胜率
  但硬币是公平的还是有问题?
  
  → 需要 ≥ 30 次才有统计意义
  → V2 在 28 月的 60 笔才是真实胜率 (55%)
```

**30 天 = 22 个交易日，6 笔交易，每笔权重 17%**。任何 1 笔异常都会扭曲胜率。

### 真实证据 — Alpha +7.63%

```
V2 实盘 30 天:
  - 我们的收益: -9.08%
  - 基准 (沪深 300) 同期: -16.71%
  - Alpha: -9.08% - (-16.71%) = +7.63%  ← 跑赢基准
  
即使胜率 6%, V2 仍少亏 7.63% — 这才是策略价值的核心。
```

### "穿越牛熊" 的真实含义

V2 不是"每天不亏"，是：
- **牛市**：赚更多（验证集年化 182%）
- **熊市**：亏更少（基准跌 20% 我们跌 13%）
- **震荡市**：少亏多赚（这次 7 月震荡就是，少亏 7.6%）

7 月不是熊市，是**震荡偏弱**（基准跌 16% 算偏弱）。V2 的应对：
- 不滥买（min_buyers=1 严格）
- 持仓止盈严格（+15% 立刻收）
- 风险可控

### 关键证据

- 状态码 200 ✅ (端点当时工作)
- `egid: 7692294622337259536` (动态签名)
- 来源页面 `https://show.jd.com/m/ZL5vVEgDqrY4lBKr/?secondDataCode=lxpydp&rankType=lxpydp`

**当前 session 调用**显示：
- 状态码 200 ✅
- `resultData.status: FAIL` (业务级失败)
- `resultData.code: 0000` (业务码成功但无数据)

**结论** — 端点本身 HTTP 200，但**业务层拒绝返回数据**。可能原因：
1. **端点是限时活动**（2026-07-12 之前的）
2. **账户无权限** (`jd_zeNbYszYHWVA` 账号没开通"投顾评级"权限)
3. **动态签名过期** (HAR 里的 `egid` 是动态生成的，不能复用)
