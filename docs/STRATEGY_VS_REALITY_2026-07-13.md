## 三、持仓对比模块

`data/my_holdings.json` 模板:
```json
{
  "asof": "2026-07-13",
  "cash": 300000,
  "holdings": [
    {"code": "005698", "name": "华夏全球科技QDII A", "shares": 1000, "cost_nav": 1.234, "entry_date": "2026-07-01"}
  ]
}
```

`compare_with_signal()` 函数:
- 加载 my_holdings
- 对每只持仓: 用 V2 规则算"该卖/该持有"
- 对每个买入信号: 与持仓去重, 给"加仓/新建/不操作"
- 输出 4 类操作: 加仓 / 止盈 / 调仓 / 观望
