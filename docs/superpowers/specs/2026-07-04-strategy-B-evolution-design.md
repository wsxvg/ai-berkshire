# 策略 B 进化版：全链路大佬追踪 + 滚动窗口进化

## 设计概要

在现有核心-卫星策略（+48.53%）基础上进化，围绕三个核心改进：
1. **全链路大佬追踪** — 区分建仓/加仓/减仓/清仓，精细化跟投
2. **滚动窗口进化** — 5轮 walk-forward 验证防过拟合
3. **半凯利仓位** — 折扣凯利+硬上限，防止激进

## 模块 1：全链路大佬追踪

追踪每个大佬对每只基金的完整操作链，输出聚合信号：

| 操作 | 权重 | 说明 |
|------|------|------|
| 建仓 | ×3.0 | 大佬首次买入，最强信号 |
| 加仓 | ×1.5 | 持有时再加，中信号 |
| 减仓 | -0.5 | 卖一部分，弱信号 |
| 清仓 | -2.0 | 全部卖掉，强卖出信号 |
| 7天超额共识 | +1.0 | 3+大佬在7天内买入同一基金 |

## 模块 2：滚动窗口进化（Walk-Forward）

```
W1: 训练 2024-03~12     → 验证 2025-01~03
W2: 训练 2024-06~2025-03 → 验证 2025-04~06
W3: 训练 2024-09~2025-06 → 验证 2025-07~09
W4: 训练 2024-12~2025-09 → 验证 2025-10~12
W5: 训练 2025-03~12     → 验证 2026-01~03
```

每个基因必须在全部 5 个窗口的验证集跑赢基准，否则淘汰。

## 模块 3：半凯利仓位

```python
半凯利 = 原凯利 × 0.5
硬上限 = min(半凯利, max_position_pct=25%, 可用现金×30%)
```
