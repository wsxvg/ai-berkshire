# 四大师风险标签层 — 改良版（P2）

基于之前的设计批评进行修正，核心改动：

## 与 V1 的关键差异

| 问题 | V1（否决） | V2（当前） |
|------|-----------|-----------|
| 评分方式 | 直接改5维基础分 (±1.5) | **不改分，只加标签** |
| 巴菲特 | 重复计分 | 合并到已有成本维度，只输出标签 |
| 李录 | 重复计分 | 合并到已有经理维度，只输出标签 |
| 芒格 | 用holder_count伪指标 | **改用板块真实成交额占比** |
| 段永平 | QDII数据滞后陷阱 | 增加数据新鲜度校验 |
| 修正幅度 | ±1.5（重锤） | ±0（不改分） |

## 四标签输出

标签在每日/季度报告中显示，**不影响评分，只提醒风险**。

```
段永平标签:
  🟢 持仓质量好（消费/科技>60%）
  🟡 高周期暴露（周期/地产>40%）  
  ⚪ 数据不足（QDII持仓滞后>2月）

巴菲特标签:
  🟢 费率优势（管理费<0.5%+C类）
  ⚪ 费率适中
  🔴 费率偏高（管理费>1.5%）

芒格标签:
  🟢 冷门赛道有机会（板块成交额占比<中位数）
  🔴 赛道过热（板块3月涨幅>30%+成交额占比创90天新高）
  ⚪ 正常

李录标签:
  🔵 经理稳定（任职>5年）
  ⚪ 一般
  🔴 经理变更（任职<1年或刚换人）
```

## 数据来源

- 板块成交额：从 fund_charts 计算（60天涨幅）
- 持仓穿透：从京东金融 holdings API
- 费率/经理：从 fund_cache 已有数据
- 数据新鲜度：对比报告日期和持仓数据日期
