# 懂车帝IPO监管与政策环境深度分析

> 研究日期：2026年5月30日
> 分析维度：监管政策与合规风险
> 研究对象：懂车帝（Dongchedi）赴港IPO所面临的全方位监管环境

---

## 核心结论

懂车帝赴港IPO的监管路径整体可行，但需要跨越多道合规门槛。相较于赴美上市，选择港股在网络安全审查层面的制度摩擦显著降低，但证监会境外上市备案、数据安全合规、VIE架构审查等环节仍需审慎应对。TikTok事件的外溢效应有限但不可忽视，字节系的政治敏感性是一个持续存在的隐性风险。

**风险等级评估：中等偏低（赴港路径下）**

## 一、中国证监会境外上市备案制度

### 1.1 制度框架

2023年3月31日起，中国证监会正式实施《境内企业境外发行证券和上市管理试行办法》及配套指引，以备案制取代原有审批制。核心要点：

- **备案时机**：企业需在向境外证券监管机构提交上市申请后3个工作日内向证监会报备
- **审核时限**：材料齐全的，证监会在20个工作日内完成审核并公示；材料不完整的，5个工作日内发出补正通知，企业须在30日内补交
- **适用范围**：直接上市和间接上市（含VIE架构）均需备案

### 1.2 备案审核的重点关注领域

根据证监会已公开的反馈意见统计，审核重点集中在以下方面：

1. **数据安全与个人信息保护**：超过一半的备案企业被问及相关合规问题，这对数据密集型的懂车帝尤为关键
2. **行业准入与外商投资限制**：TMT行业是重点审查领域，需确认是否涉及外商投资准入负面清单
3. **VIE架构合规性**：要求完整披露VIE安排、相关风险及应对措施
4. **国家安全审查**：禁止可能危害国家安全的境外上市行为

### 1.3 懂车帝的备案进展

截至目前，公开渠道尚未查到懂车帝在证监会的具体备案记录。据2026年2月至3月的媒体报道，懂车帝已选定花旗和高盛作为IPO承销商，计划最早2026年在港股上市，募资10亿至15亿美元。这意味着正式备案流程可能正在推进中，或即将启动。

### 1.4 备案环境参考数据

- 截至2025年底，共有179家企业获得证监会境外上市备案通知，较2024年增长15%
- 2026年3月，29家企业提交新申请，全部选择香港上市
- 2026年4月，17家企业完成备案，其中16家赴港、1家赴纳斯达克
- 赴港上市已成为绝对主流方向，监管态度相对友好

## 二、数据安全法与个人信息保护法的影响

### 2.1 懂车帝的数据敏感性分析

懂车帝作为汽车信息与交易平台，涉及的数据类型包括：

| 数据类型 | 敏感等级 | 说明 |
|---------|---------|------|
| 用户浏览/搜索行为 | 中 | 反映消费偏好，属于个人信息 |
| 用户注册信息 | 中高 | 手机号、姓名等基本个人信息 |
| 汽车交易数据 | 中 | 购车意向、价格信息、经销商数据 |
| 内容创作者数据 | 中 | 约750万创作者的身份与行为数据 |
| 汽车技术测评数据 | 低 | 车辆性能参数，属于商业数据 |
| 位置信息 | 高 | 涉及附近经销商推荐等功能 |

**关键数据规模**：
- 日活用户超过1000万
- 汽车兴趣用户超过5亿
- 内容创作者约750万
- 服务经销商超过3万家

### 2.2 适用法规矩阵

**《数据安全法》（2021年9月实施）**：
- 要求数据分类分级管理
- 涉及"重要数据"出境需经安全评估
- 汽车行业数据可能涉及行业主管部门认定的"重要数据"范畴

**《个人信息保护法》（2021年11月实施）**：
- 处理超过100万人个人信息的处理者，向境外提供个人信息前须经国家网信办安全评估
- 懂车帝用户规模远超此门槛

**《网络数据安全管理条例》（2025年1月实施）**：
- 对上位法相关制度予以细化补充
- 明确数据处理者的分类义务

**《汽车数据出境安全指引（2025版）》（征求意见稿）**：
- 八部门联合起草，明确汽车数据出境的九大豁免场景
- 累计向境外提供100万人以上个人信息的，须申报数据出境安全评估
- 关键信息基础设施运营者向境外提供个人信息的，同样须申报

### 2.3 对懂车帝的实质影响

懂车帝选择港股上市的一个关键优势：**赴港上市本身不直接触发数据出境安全评估的义务**。但以下情形仍需关注：

1. 如上市后涉及境外投资者/审计机构获取用户数据，可能触发数据出境要求
2. 招股书中需充分披露数据合规措施
3. 需证明已建立完善的数据分类分级制度

## 三、网络安全审查要求

### 3.1 核心规定

《网络安全审查办法》（2022年2月15日施行）的关键条款：

- **强制申报条件**：掌握超过100万用户个人信息的网络平台运营者**赴国外上市**，必须在向外国证券监管机构提出上市申请之前申报网络安全审查
- **审查评估重点**：关键信息基础设施被非法控制的风险；核心数据、重要数据或大量个人信息被窃取、泄露、非法出境的风险；上市存在被外国政府影响、控制、恶意利用的风险

### 3.2 懂车帝是否需要网络安全审查？

**关键区分：赴港 vs 赴美**

2022年修订的《网络安全审查办法》明确规定，强制申报网络安全审查的触发条件是**"赴国外上市"**。在2022年的修订过程中，立法者特意删除了早期《网络数据条例》中"数据处理者赴香港上市，影响或者可能影响国家安全的"这一情形。

**结论：懂车帝赴港上市，不自动触发网络安全审查的强制申报义务。**

但需注意两个例外情形：
1. 如果国家网信办主动认定懂车帝的产品或服务"影响或者可能影响国家安全"，仍可依职权发起审查
2. 如果懂车帝未来考虑在美国等"国外"市场上市或发行存托凭证，则必须申报

### 3.3 懂车帝已有的合规动作

- 懂车帝已通过国家信息安全等级保护（三级）测评和备案
- 2026年4月，懂车帝的汽车垂类大模型通过国家网信办的生成式AI服务备案（重庆市网信办公示）
- 这些合规动作表明公司在主动为IPO扫清监管障碍

## 四、字节跳动的VIE结构与懂车帝的持股架构

### 4.1 分拆历程

懂车帝的独立化进程分为几个关键节点：

| 时间 | 事件 |
|------|------|
| 2017年 | 懂车帝App上线，隶属于字节跳动旗下 |
| 2023年10月 | 重庆懂车帝科技有限公司成立，作为App运营主体 |
| 2023年12月 | 厦门懂车族科技有限公司成立，成为新的控股股东 |
| 2023年12月21日 | 北京懂车帝科技有限公司完成工商变更，今日头条退出股权架构 |
| 2024年6月-10月 | 完成首轮独立融资约6亿-8亿美元，估值30亿美元 |
| 2026年2-3月 | 传选定花旗、高盛为IPO承销商 |

### 4.2 当前境内持股架构

```
厦门懂车族科技有限公司（100%持股）
├── 何戬（懂车帝总裁）：60%
└── 赵静：40%
    │
    └── 北京懂车帝科技有限公司（境内运营主体）
         │
         ├── 重庆懂车帝科技有限公司（App运营主体，全资子公司）
         └── [质押给] 北京懂车族科技有限公司
```

### 4.3 境外架构线索

根据公开信息：
- 北京懂车帝的穿透股权显示，其全资由**香港懂车帝科技有限公司**（原"香港幸福里科技有限公司"，2021年成立）持有
- 这表明懂车帝已搭建了典型的VIE/协议控制架构：**境外上市主体（预计为开曼公司）→ 香港中间层公司 → 境内VIE协议控制 → 境内运营实体**

### 4.4 字节跳动的实际控制权

虽然表面上今日头条已退出懂车帝的直接股权架构，但字节跳动通过以下机制保持实质控制：

1. **股权质押**：厦门懂车族将北京懂车帝的股权质押给北京懂车族科技有限公司（预计与字节跳动存在关联）
2. **管理层安排**：何戬作为懂车帝总裁，本身是字节系高管
3. **业务依赖**：懂车帝的内容分发深度依赖抖音生态，字节跳动通过流量和技术维持实质影响力
4. **历史投资者重合**：红杉中国、KKR、泛大西洋投资既是字节跳动股东，也是懂车帝投资者

### 4.5 VIE架构在证监会备案中的审查要点

根据现行规定，VIE架构企业境外上市备案需特别关注：

- VIE协议的完整性和合法性
- 协议控制是否存在被认定为无效的风险
- 涉及的行业是否属于外商投资准入负面清单
- 发行人对VIE实体的实际控制力证明

**懂车帝面临的特殊问题**：汽车信息服务是否属于增值电信业务？如属于，则涉及外商投资准入限制，VIE架构的必要性和合规性将成为审查重点。

## 五、港交所上市要求分析

### 5.1 可选上市路径

懂车帝赴港上市理论上有两条路径：

**路径一：主板常规上市（第8章）**

盈利测试要求：
- 最近一年盈利不少于3500万港元
- 前两年累计盈利不少于4500万港元
- 三年营业记录

市值/收入测试（替代盈利测试）：
- 上市时市值不低于40亿港元
- 最近一个经审计会计年度收入不低于5亿港元

**路径二：特专科技公司上市（第18C章）**

- 已商业化公司：最低市值40亿港元（2024年9月修订后，原为60亿，有效期至2027年8月）
- 未商业化公司：最低市值80亿港元（原为100亿）
- 研发投入要求：已商业化公司研发支出占总营运开支至少15%
- 覆盖行业：包括电动及自动驾驶汽车等先进硬件领域

### 5.2 懂车帝最可能的选择

**预判：主板常规上市（市值/收入测试），而非18C章。**

理由：
1. 懂车帝的估值约30亿美元（约234亿港元），远超40亿港元门槛
2. 懂车帝作为汽车信息与交易平台，核心业务是广告和线索服务，属于互联网平台而非"特专科技"
3. 如已实现盈利，可走盈利测试；如未盈利，可走市值/收入测试
4. 汽车之家（同业对标）2021年以常规方式回归港股，是有效先例

### 5.3 港交所的审核关注点

对懂车帝这类互联网平台公司，港交所可能重点关注：

1. **业务可持续性**：对字节跳动/抖音生态的依赖程度，独立运营能力
2. **关联交易**：与字节跳动之间的流量采购、技术服务等关联交易的定价公允性
3. **竞业限制**：字节跳动是否承诺不从事竞争业务
4. **数据合规披露**：中国数据安全法规对业务的影响
5. **控制权安排**：VIE架构的风险披露

### 5.4 科企专线新机制

2025年5月6日起，港交所与证监会联合推出"科企专线"机制：
- 允许特专科技公司和生物科技公司在正式申请上市前获得专门指引
- 可选择以保密形式提交上市申请
- 降低了企业与监管机构之间的沟通成本

## 六、TikTok事件对字节系上市的潜在影响

### 6.1 TikTok事件回顾

| 时间 | 事件 |
|------|------|
| 2024年4月 | 拜登签署"不卖即禁"法案 |
| 2025年1月17日 | 美国最高法院裁定该法不违宪 |
| 2025年1月18-19日 | TikTok短暂停止美国服务后恢复 |
| 2025年1-9月 | 特朗普多次延长宽限期 |
| 2025年9月 | 马德里谈判达成初步共识 |
| 2026年1月 | 甲骨文和银湖资本牵头的中美合资企业成立，负责TikTok美国业务运营，字节跳动持有19%股份 |

### 6.2 对懂车帝IPO的影响评估

**直接影响：有限**

1. TikTok问题已通过合资方案初步解决，最大不确定性消除
2. 懂车帝是纯中国境内业务，不涉及美国用户数据
3. 懂车帝选择港股（而非美股），进一步隔离了美国监管风险

**间接影响：需持续关注**

1. **品牌关联风险**：国际投资者可能对"字节系"公司存在偏见或审慎态度，认为中国科技公司面临地缘政治风险
2. **估值折价可能**：字节跳动的政治敏感性可能导致懂车帝IPO估值承受一定折价
3. **美国投资者参与度**：部分美国机构投资者可能受到内部合规限制，减少对字节系公司的投资
4. **制裁连锁风险**：虽然概率极低，但若中美关系进一步恶化，不排除美国将制裁扩展至字节跳动其他业务的可能
5. **数据安全叙事的强化**：TikTok事件使全球监管机构对中国科技公司的数据安全关注度提升，懂车帝在路演中需要更详细地回应数据安全问题

### 6.3 字节跳动整体上市的信号意义

懂车帝作为字节系首个独立上市实体，具有"探路"意义：

- 如果懂车帝成功上市，将验证字节系公司通过分拆方式实现资本化的路径
- 为字节跳动其他业务线（如飞书等）的后续上市提供经验和信心
- 投资者对懂车帝IPO的态度，某种程度上是对字节跳动品牌在资本市场信任度的一次检验

## 八、投资者视角的关键问题

### 监管不确定性定价

对于考虑参与懂车帝IPO的投资者，以下监管问题值得在尽职调查中重点关注：

1. **证监会备案是否已获受理/通过？** 这是上市的前置条件
2. **懂车帝与字节跳动的关联交易规模和定价机制如何？** 影响利润真实性
3. **字节跳动对懂车帝的实际控制权安排是否透明？** 影响治理结构评价
4. **数据合规成本有多大？** 可能影响未来利润率
5. **抖音流量供给是否有长期保障？** 决定了业务独立性和估值可持续性
6. **汽车垂类大模型的合规备案是否完整？** AI监管是新增变量

### 对标参考

汽车之家2021年从美股回归港股二次上市，当时港股市值一度接近千亿港元，目前约240亿港元。懂车帝30亿美元（约234亿港元）的Pre-IPO估值与汽车之家当前市值基本相当，但能否在IPO中获得溢价，很大程度上取决于监管合规的完整性和增长故事的说服力。

## 九、结论与展望

### 核心判断

1. **监管路径可行**：懂车帝选择港股上市是当前最优选择，避开了赴美上市可能触发的网络安全审查，证监会备案制度也已趋于成熟和可预期
2. **合规准备充分**：从已通过信息安全等保三级认证、AI大模型备案等动作看，公司在主动推进合规建设
3. **VIE架构是关键审查点**：字节跳动通过质押等机制保持实控权的安排，在证监会备案中将被重点关注
4. **TikTok影响有限但存在**：直接风险已基本消除，但"字节系"的政治标签可能带来估值层面的隐性折价

### 未来需要跟踪的监管节点

- 证监会境外上市备案的正式受理与审核进展
- 招股书提交港交所的时间节点
- 汽车数据出境安全指引（2025版）的正式出台及对懂车帝的适用情况
- 中美关系走向及对字节系公司的溢出效应
