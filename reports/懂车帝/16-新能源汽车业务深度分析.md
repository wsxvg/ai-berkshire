# 懂车帝新能源汽车业务深度分析

> 研究日期：2026年5月
> 分析维度：新能源转型机遇、渠道变革、内容生态、竞争格局

---

## 一、新能源汽车在懂车帝平台的内容与交易占比

### 1.1 市场大背景：新能源已成绝对主流

中国新能源汽车渗透率在2025年全年达到53.9%，年度渗透率首次超过燃油车。到2026年4月，新能源渗透率历史性突破60%，懂车帝发布的全国乘用车零售销量排行榜Top 10中仅剩一款燃油车型，其余九席被新能源车包揽。

中汽协预测2026年新能源汽车销量将达1900万辆，占汽车总销量的比重持续攀升。

### 1.2 懂车帝平台内容结构的新能源转向

根据懂车帝发布的多份报告和平台数据：

- **用户意向偏移**：懂车帝DAU持续增长，平台聚集大量高浓度购车人群，其中新能源意向用户占比已处于高位。女性用户更倾向接受新能源（尤其纯电），新能源用户在高线城市更集中。
- **内容生态重心转移**：新能源相关内容热度飙升，车机、充电、辅助驾驶等专有名词搜索热度同比上涨27%。懂车帝已明确将新能源作为内容扶持的核心方向。
- **创作者结构变化**：科技、财经、数码等非传统汽车类作者持续进入新能源内容领域，不少新能源车主实现从用户到KOC的身份转化。
- **交易维度**：懂车帝服务以旧换新带动汽车消费规模接近3800亿元（2025年企业社会责任报告数据），新能源是以旧换新的核心驱动品类。

**关键判断**：虽然懂车帝未公开披露新能源内容的精确占比数字，但从用户行为数据、内容搜索热度、销量排行榜结构来看，新能源内容占比大概率已超过50%，且仍在快速提升。这与整体市场渗透率的趋势高度一致。

### 1.3 用户行为差异：燃油车用户vs新能源用户

懂车帝2025年4月发布的用户行为报告揭示了一个值得关注的差异：

| 维度 | 自主燃油用户 | 新能源用户 |
|------|------------|-----------|
| APP依赖度 | 更高（强工具属性） | 相对较低 |
| 内容消费 | 以选车为主 | 更沉浸内容消费及车友圈社交 |
| 价格带集中 | 5-15万占82% | 5-15万占58.1%（更分散） |
| 用户画像 | 低龄段未婚更多 | 高线城市更集中 |

**这意味着什么**：新能源用户对懂车帝作为"工具型"平台的依赖度略低于燃油车用户。新能源用户可能更多通过品牌官方渠道、社交媒体等途径获取信息。这对懂车帝的内容策略提出了更高要求——需要从纯工具型平台升级为"内容+社交+交易"的综合生态。

## 二、新能源车企在懂车帝的投放与合作情况

### 2.1 主要新能源车企与懂车帝的关系

**比亚迪**：关系经历波折。2023年懂车帝推动CPS（按成交付费）模式切换时，因线索费用大幅上涨（经销商反映涨幅达三四倍），比亚迪发布内部邮件要求旗下所有门店停购懂车帝年框。这是对懂车帝商业模式的一次重大冲击——比亚迪是中国最大的新能源车企，其态度具有行业风向标意义。后续双方关系有所修复，但这一事件暴露了懂车帝在面对头部车企时的议价弱势。

**华为/鸿蒙智行**：2023年冬测争议后，华为与懂车帝、汽车之家、易车三大平台均暂停合作，引发行业震动。2025年7月的智驾测试中，鸿蒙智行对懂车帝测试结果回应"不予置评"，双方关系仍较敏感。

**蔚来/小鹏/理想/小米**：这些品牌均参与了懂车帝的各类测试项目（冬测、智驾测试等），但其中小米SU7在2025年智驾测试中表现不佳引发争议。总体而言，新势力车企对懂车帝保持合作态度，但对测试结论有时持保留意见。

**特斯拉**：在懂车帝2025年智驾测试中，特斯拉Model 3/Model X以纯视觉方案拿下综合排名前两位，马斯克亲自转发测试结果。这一定程度上提升了懂车帝测试的国际影响力。

### 2.2 懂车帝与车企的合作模式

1. **线索/流量合作**：车企和经销商向懂车帝购买销售线索，这是懂车帝核心收入来源。
2. **品牌广告投放**：开屏广告、信息流广告、品牌专区等。
3. **内容合作**：包括懂车帝测试项目的车辆提供、联合测评等。
4. **达人营销**：通过"懂车星选"产品，在懂车帝和抖音双端进行达人营销投放。
5. **以旧换新政策落地**：懂车帝作为官方授权平台，为车企提供补贴申请服务。

### 2.3 关键矛盾：测评独立性 vs 商业合作

懂车帝面临一个结构性矛盾：

- **作为内容平台**：需要通过独立、严苛的测试（冬测、智驾测试）建立公信力和用户信任
- **作为商业平台**：需要维护与车企的合作关系以获取广告和线索收入

2023年冬测引发华为等多家车企质疑，2025年智驾测试再次引发行业争议。这表明懂车帝在"做裁判"和"做生意"之间的平衡难度很大。车企有时会以暂停合作作为施压手段。

## 三、新能源直营模式 vs 经销商模式对懂车帝线索业务的影响

### 3.1 行业渠道变革：直营"退烧"，经销商回归

2025年第四季度，新能源车企的渠道模式迎来集中变革：

- **腾势**：在河北、河南、浙江、广东等核心市场，推进直营店关店或转让给经销商
- **鸿蒙智行**：启动大范围直营网络裁撤，广东、四川、江苏等省份渠道调整全面落地
- **理想、蔚来**：此前坚守直营的代表，也开始探索更具成本效益的渠道优化路径

**核心驱动因素**：一线城市核心商圈直营展厅年租金百万元，叠加人员、水电物业等支出，单店年运营成本达五六百万元。200家门店规模的车企，仅开店和运营就需年投入约10亿元。在利润持续承压的环境下，这一负担难以为继。

行业正加速向"直营+加盟/授权"混合模式转型：直营保留一线核心商圈的品牌展示功能，加盟和授权渠道负责快速下沉三四线及县域市场。

### 3.2 对懂车帝线索业务的影响

**利好层面**：

1. **经销商回归扩大线索需求**：直营模式下，车企自建线索获取体系，对第三方平台依赖较低；经销商模式下，经销商对外部线索的需求天然更强。新能源车企回归经销商体系，意味着更多门店需要购买懂车帝线索。
2. **下沉市场扩容**：低线城镇汽车销量及增速远超一二线城市，经销商模式更适合下沉市场覆盖，这扩大了懂车帝的潜在线索客户池。
3. **CPS模式更契合经销商利益**：懂车帝推动的CPS（按成交付费）模式，经调整优化后，2024年数据显示57%的合作经销商成交量正增长，94.8%的经销商愿意继续合作。

**风险层面**：

1. **头部车企自建流量闭环**：比亚迪、理想等头部车企通过创始人IP、品牌社交媒体等方式自建获客渠道，降低对懂车帝的依赖。李斌、李想、雷军等造车大佬打造个人IP，善用抖音、小红书、视频号等平台直接获客。
2. **线索定价博弈**：CPS模式切换曾引发比亚迪等大客户抵制。懂车帝在提升线索单价的同时面临客户流失风险。
3. **新能源车的在线化购买趋势**：部分新能源品牌支持线上直接下单，减少了对传统"留资-跟进-到店"线索链路的依赖。

**净影响判断**：短期来看，直营转经销的趋势对懂车帝线索业务是利好。但中长期看，头部车企自建流量体系的趋势不可逆，懂车帝需要从"卖线索"升级为"帮成交"，才能维持线索业务的价值。

## 四、懂车帝冬测、续航测试等新能源评测内容的影响力

### 4.1 冬测：从国内到全球的升级路径

**2023年冬测（争议之年）**

- 问界M7增程版续航达成率仅31.6%排名倒数第一，引发华为余承东公开质疑
- 吉利、长城旗下多款车型续航达成率不到40%
- 问界官方指出，测试前车辆进行了71分钟空调制热，期间十余次开关门窗，严重偏离真实使用场景
- 华为随后暂停与三大汽车垂直平台的合作

**2025年全球冬测（升级之年）**

- 升级为全球测试：在中国黑河、美国阿拉斯加、瑞典于默奥三地进行
- 覆盖近40个品牌、90多辆车
- 测试科目扩展至基础用车、主动安全、极限续航、拉力计时等
- 累计行驶近10万公里
- 获央视关注和报道，国际化提升品牌影响力

### 4.2 智驾安全测试：2025年最具影响力的评测事件

2025年7月，懂车帝与央视新闻合作，包下15公里真实高速公路及城市道路，对36款主流车型进行辅助驾驶安全模拟测试：

- **高速场景**：6类场景，36款车，183次测试，通过44次，**通过率仅24%**
- **城市场景**：9类场景，26款车，233次测试，通过103次，**通过率44.2%**
- 综合排名：特斯拉Model 3第一，Model X第二，蓝山、小鹏G6、问界M9分列三至五位

这次测试引发巨大社会反响：
- 马斯克亲自转发并表示认可
- 鸿蒙智行回应"不予置评"
- 公安部交通管理局借此明确表态：当前"智驾"均为辅助驾驶，非自动驾驶
- 部分用户和车企质疑极端场景测试的代表性

### 4.3 评测影响力评估

| 维度 | 评估 |
|------|------|
| 公众关注度 | 极高，多次登上热搜，马斯克转发提升国际影响力 |
| 行业影响力 | 高，但争议也大，多家车企质疑方法论 |
| 政策影响力 | 有一定作用，推动公安部对智驾的明确表态 |
| 用户信任度 | 存在分化，部分用户认可其独立性，部分认为测试标准不公 |
| 竞争壁垒 | 高，冬测/智驾测试已成为懂车帝的标志性IP内容 |

**关键判断**：冬测和智驾测试已成为懂车帝最重要的内容IP和品牌资产。它们带来了巨大的流量和话题度，但也带来了与车企关系的紧张。对于IPO前的懂车帝来说，这些评测内容是展示平台价值的"杀手锏"，但也需要在方法论的科学性和透明度上持续改进，以维护公信力。

## 五、新能源车企营销预算的渠道流向

### 5.1 行业营销费用概览

2025年第一季度中国车企销售费用排名中，小米超越比亚迪位居第一，反映出新入局者在品牌建设上的激进投入。不同车企的营销策略分化明显：

- **理想**：声称营销费用率全行业最低，接近特斯拉水平。依赖创始人IP和口碑传播。
- **小米**：高投入策略，利用雷军个人影响力+多渠道饱和攻击。
- **蔚来/小鹏**：营销费用占比处于高位，但相较此前有明显下降趋势。
- **比亚迪**：销量规模大但营销投入相对克制，更多依赖产品力和价格竞争力。

### 5.2 渠道投放格局的结构性变化

**传统格局被颠覆**：

传统上，汽车之家、懂车帝、易车三大垂直平台是车企线上营销的核心渠道。但这一格局正在被打破：

1. **抖音/短视频崛起**：造车大佬（李斌、李想、雷军、余承东）打造个人IP，在抖音直接获客。车企品牌官方号也成为重要的传播和获客渠道。
2. **小红书种草**：在女性用户和年轻群体中的影响力日增，成为新能源车企（尤其中高端品牌）的重要投放阵地。
3. **视频号/微信生态**：私域运营、社群裂变成为车企降低获客成本的重要方式。
4. **品牌自有渠道**：车企APP、小程序、线下活动等自有渠道的投入加大。

**垂直平台的压力**：

车企和经销商已经开始把预算从汽车之家、懂车帝等垂直平台转向抖音、小红书等"新势力"渠道。车企在减少对垂直平台依赖的同时，实现自有销售线索的大量获取和快速转化。

### 5.3 懂车帝能占到多少？

懂车帝没有公开披露具体的广告收入数据。从估值和融资信息推断：

- 2024年融资约6亿美元，估值约30亿美元
- 2026年传闻IPO，拟募集10-15亿美元
- 截至2025年8月，DAU突破1000万，覆盖3万+经销商、110+品牌

**懂车帝的独特优势**：

背靠字节系，懂车帝拥有与抖音双端联动的独特能力。数据显示，用户在抖音端激发兴趣后，约84%会跳转至懂车帝查看车型库，78%搜索具体车系并阅读深度评测。这种"兴趣激发(抖音) -> 深度决策(懂车帝) -> 交易转化"的链路是汽车之家和易车不具备的。

**份额估判**：在新能源车企数字营销预算中，懂车帝（含抖音联投）可能占到15%-25%的份额，但这一比例面临来自品牌自建渠道和新兴平台的持续挤压。

## 六、智能化（智驾/座舱）内容是否成为新增长点

### 6.1 智驾评测：已成为懂车帝2025年最大的内容增长点

2025年7月的智驾安全实测是懂车帝历史上影响力最大的内容事件之一：

- 与央视新闻联合制作，提升权威性
- 测试覆盖36款车、15类高危场景、数百次模拟碰撞
- 马斯克亲自转发，引发全球关注
- 全网曝光量巨大

懂车帝已明确表态将"加大智能化、安全性实测投入"，并推出"百万车主评测计划"，将智能化内容作为平台差异化竞争的核心方向。

### 6.2 AI赋能平台功能：从工具到生态

**2025年：AI选车上线**

懂车帝首次公开AI全景图，基于平台5000万+结构化数据、7.6万+车型库、4000+款车实测数据（超100万条），自主研发AI大模型，形成选车、比车、买车、估价等多元场景的AI智能体。

**2026年北京车展：AI小懂首发**

懂车帝在2026年北京车展正式发布AI助手"AI小懂"，从"选车"单一场景升级至"选、买、卖、用、玩"五大汽车生活全环节。核心能力包括：
- 高精度智能交互
- 专属汽车知识库
- 深度多轮对话
- 海量真实用户评价整合

**与火山引擎的技术协同**：懂车帝与字节旗下火山引擎合作，打造更智能的汽车内容生态。

### 6.3 增长点评估

智能化内容确实是懂车帝的新增长点，但需要区分两个层面：

1. **内容维度**：智驾评测等内容带来大量流量和话题度，但难以直接货币化，且可能得罪广告客户（车企）
2. **产品维度**：AI选车、AI小懂等智能化功能提升了用户体验和平台粘性，有助于延长用户生命周期价值

**潜在风险**：智驾评测的争议性可能导致部分车企减少投放。懂车帝需要在内容影响力和商业利益之间找到平衡。

## 七、充电桩/补能相关服务布局

### 7.1 懂车帝充电平台

懂车帝已推出充电公共服务平台，具体布局如下：

- **重庆试点**：与"渝易充"App通过市场化合作实现互联互通，在重庆覆盖1000余个充电站、9000余个充电桩、近1.4万个充电枪
- **全国扩展**：服务已扩展至31个省份的324个城市，覆盖主流充电品牌
- **功能**：实时查询充电站距离、价格、停车费等信息，支持一键扫码、跨平台使用

### 7.2 定位与策略

懂车帝在补能服务上采取的是**轻资产聚合模式**——不自建充电桩，而是聚合第三方充电服务，为用户提供一站式查询和支付体验。这与其"汽车生活平台"的定位一致。

### 7.3 业务价值评估

| 维度 | 评估 |
|------|------|
| 战略价值 | 高。延伸用户服务链条，从"买车"延伸到"用车"，提升用户粘性和LTV |
| 收入贡献 | 目前较小。充电聚合业务竞争激烈（高德、百度地图等也在做），利润空间有限 |
| 竞争壁垒 | 低。充电聚合本质是信息服务，技术壁垒不高 |
| 与主业协同 | 中等。可以为新能源用户提供一体化体验，但与核心的线索/广告业务关联度有限 |

**判断**：充电服务是懂车帝"新能源生态"拼图的一块，更多是完善用户体验的补充业务，短期内不会成为重要收入来源。但随着新能源保有量持续增长，这一业务的战略价值会逐步显现。

## 八、综合判断与关键结论

### 8.1 新能源对懂车帝的影响：机遇大于挑战，但有结构性风险

**核心机遇**：
1. 新能源渗透率突破60%且仍在上升，懂车帝平台内容和交易生态自然受益
2. 新能源车企从直营回归经销商模式，扩大了线索业务的客户基础
3. 智驾评测等新内容IP创造了差异化优势和流量增长点
4. 抖音+懂车帝双端联动形成独特的"兴趣激发-深度决策-交易转化"闭环

**结构性风险**：
1. 头部新能源车企（比亚迪、理想等）自建流量体系，降低对第三方平台的依赖
2. 评测独立性与商业利益的冲突可能激化——车企用暂停合作施压
3. 新能源用户对懂车帝APP的依赖度低于燃油车用户，平台粘性面临挑战
4. 营销预算从垂直平台向社交媒体和品牌自有渠道持续转移

### 8.2 懂车帝的差异化壁垒

相比汽车之家和易车，懂车帝在新能源时代的核心壁垒包括：

1. **字节系生态协同**：抖音导流 -> 懂车帝转化，84%的抖音汽车兴趣用户会跳转懂车帝
2. **评测内容IP**：冬测和智驾测试已形成强品牌认知，央视背书提升权威性
3. **AI技术能力**：依托火山引擎，AI选车/AI小懂等产品领先同行
4. **数据积累**：结构化数据5000万+，实测数据库覆盖4000+款车

### 8.3 对IPO估值的影响

懂车帝正筹备2026年港股IPO，拟募资10-15亿美元。新能源业务的表现将直接影响投资者对其成长性的判断：

- **有利因素**：新能源渗透率持续提升带来行业顺风；全球冬测和智驾评测展示了内容壁垒；AI产品矩阵体现技术能力
- **不利因素**：与头部车企的关系波动增加收入不确定性；线索业务可能面临价格天花板；充电等新业务短期贡献有限

### 8.4 值得持续跟踪的关键指标

1. 懂车帝DAU中新能源意向用户占比的变化趋势
2. 新能源车企（尤其比亚迪、华为）与懂车帝合作关系的修复情况
3. CPS模式下经销商的续约率和客单价变化
4. 冬测/智驾测试等内容IP的商业化路径探索
5. AI选车/AI小懂的用户渗透率和对转化效率的提升效果
6. 充电服务的MAU和交易规模增长
