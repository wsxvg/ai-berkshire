# PDD研究覆盖度评估：信息缺口分析

> 方法：对照网上最新公开信息，逐一核对现有60+份研究文件的覆盖范围

## 一、明确缺失的专题

### 1. 多多买菜专题研究（重要性：高）

社区团购"一家独大"格局是2025-2026年重大竞争格局变化：

- 美团优选退出、京喜拼拼边缘化、淘菜菜收缩，多多买菜成为唯一全国性玩家
- 2025年GMV接近3000亿元，超2024年多多买菜与美团优选GMV总和
- 美团优选撤出后日单量暴涨50%，全国72个仓库全速运转
- 多多买菜暂时放弃即时零售、放下短期盈利目标，全力抢占美团退出后的市场空白
- 未来数年增速预计20%-30%，市场前景有望突破5000亿元，利润可达数十亿甚至百亿级

**现有文件**："新业务专题"（2026-04-10）将多多买菜作为子话题涉及，但缺乏对竞争格局剧变、盈利前景、估值贡献的独立深度拆解。

### 2. 免费送村/末端物流战略（重要性：中高）

千亿扶持计划的核心组成部分，对PDD下沉市场渗透意义重大：

- 2025年Q4试点，2026年全面推广
- 模式：商家发货至县级中转仓→第三方物流配送至村级取货点→平台全额承担二段运费
- 响应中央一号文件"深化快递进村"指导
- 全国多地已建成县级中转仓+村级取货点网络（如陕西扶风县71个站点）
- 2025年农村网络零售额首次突破3万亿元
- 双向价值：打通工业品下乡最后一公里 + 农产品上行最初一公里

**现有文件**：无。

### 3. 港股二次上市进展追踪（重要性：中高）

多份研究文件提到"等待港股上市"作为核心催化剂，但缺乏进展追踪：

- PDD已更换香港审计机构，聘请安永香港审计师审查财务报表（取代北京审计师），暗示推进港股上市
- 高盛估计27只中概股（总市值1.35万亿元）有资格在香港双重主要上市或第二上市，PDD位列其中
- 港股上市使ADR可转换为港股，对冲美国退市风险
- 港交所正在进行上市改革咨询，涉及同股不同权、二次上市等制度
- 尚无官方时间表

**现有文件**：无独立文件。在买入风险全景分析中简要提及。

### 4. 百亿超市/即时零售布局（重要性：中）

PDD首次真正试探即时零售赛道的新业务信号：

- 2026年初内测"百亿超市"，强化快消品优势品类
- 采用"轻资产+众包配送"策略，复用社区团购网格仓降低前期投入
- 上海等一线城市试点自建前置仓，计划逐步扩展
- 竞争对手实力强劲：京东超市用户+18%/订单量+20%，美团小象前置仓覆盖30+城市，阿里系多维度商超矩阵
- PDD因外卖大战中其他玩家的惨痛代价，明确不做餐饮外卖，聚焦非餐饮闪送

**现有文件**：无。

### 5. 机构持仓全景（重要性：中）

现有文件仅覆盖网易13F和段永平持仓，缺少整体机构视角：

- 611家机构持有PDD，共计4.24亿股
- 最新季度：262家加仓 vs 308家减仓（净减仓方向）
- 最大持仓机构：BlackRock、Vanguard、Baillie Gifford、FMR（富达）、State Street、Hillhouse
- 分析师共识评级"买入"，但目标价从$144下调至$120
- BNP Paribas以"Underperform"评级+$89目标价发起覆盖
- 麦格理从"跑赢"下调至"中性"，目标价从$151砍至$87

**现有文件**：网易13F-PDD持仓追踪、段永平PDD持仓全历史。缺整体机构持仓分析。

## 四、建议优先级

按研究价值和紧迫性排序：

1. **多多买菜专题** — 竞争格局剧变，对PDD国内估值贡献的判断直接影响买入决策
2. **港股二次上市追踪** — 催化剂进展，影响持仓信心和时间预期
3. **免费送村战略** — 理解PDD"重仓中国供应链"战略的关键拼图
4. **机构持仓全景** — 了解聪明钱的态度变化
5. **百亿超市/即时零售** — 新业务布局，目前仍在内测早期，紧迫性较低
6. **法律风险汇编** — 整合分散信息，买入风险文件已覆盖大部分核心内容
