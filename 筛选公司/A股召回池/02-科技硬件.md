## 1. 中际旭创（300308）— 光模块全球龙头

### 基本信息

| 指标 | 数据 |
|------|------|
| 市值 | 约9500-9800亿元（股价约880元） |
| PE(TTM) | 约64倍 |
| 股息率 | 0.16% |

### 生意质量 ★★★★

商业模式相对清晰：为全球超大规模数据中心提供高速光模块，本质是"卖铲子"给AI基建。收入随AI资本开支增长而增长，客户是谷歌、微软、Meta等云巨头。2025年净利润约150亿元（估计），2026年机构预期230-260亿元，增速极快。

但需注意：光模块本质仍是制造业，技术迭代快（400G→800G→1.6T），每一代产品生命周期约2-3年，需要持续研发投入保持领先。并非真正的"一次投入、持续收割"的简单生意。毛利率30%+在硬件制造中算优秀，但远不如软件或平台型公司。

### 护城河强度 ★★★★

- **技术领先**：800G市场份额超50%，1.6T领先同业6-12个月，"量产一代、研发一代、预研一代"节奏
- **客户绑定**：深度嵌入谷歌OCS生态，作为光模块核心供应商，客户转换成本高
- **规模效应**：全球最大光模块厂商，产能规模和良率控制形成成本优势
- **但另一方面**：新易盛等竞争对手在快速追赶，光模块行业历史上出现过龙头更替（如Finisar被II-VI收购），技术壁垒不是不可逾越

### 管理层质量 ★★★★

实控人王伟修（享国务院特殊津贴），2016年收购苏州旭创后主动让位，交由创始人刘圣（中科院硕士+佐治亚理工博士）主导经营。形成"技术专家+职业经理人"的治理结构。管理层在战略判断上表现优秀——2016年收购旭创是决定性一步，近年在CPO、硅光等前沿技术上持续布局。

### 估值吸引力 ★★

当前PE(TTM)约64倍，对应2026年前瞻PE约40-42倍。考虑到利润增速120%-150%，PEG约0.3-0.5，看起来"不贵"。但问题是：AI资本开支的周期性风险、光模块行业历史上的周期波动（2018-2019年曾大幅下滑）、以及当前万亿市值是否已充分反映了乐观预期。市场给予了极高的增长确定性溢价。

### 综合评分 ★★★★

### 一句话总结

AI光模块赛道的绝对王者，技术领先+客户绑定构成强护城河，但万亿市值已隐含极高增长预期，适合在行业出现恐慌或增速预期下修时关注。

## 2. 中微公司（688012）— 刻蚀设备

### 基本信息

| 指标 | 数据 |
|------|------|
| 市值 | 约2000-2100亿元（股价约330元） |
| PE(TTM) | 约93倍（静态），2026E约64倍 |
| 股息率 | 0.09% |

### 生意质量 ★★★★

半导体设备是典型的"卖铲子给淘金者"模式。客户一旦验证通过设备，替换成本极高（需要重新验证工艺，耗时数月至一年）。刻蚀设备是芯片制造三大核心环节之一，市场空间大、技术壁垒高。2025年营收约124亿元（同比+37%），净利润约21亿元（同比+31%），增长稳健。

但需注意：半导体设备行业具有明显周期性，受下游晶圆厂资本开支驱动。且中微在全球范围仍面临泛林（Lam Research）、东京电子等巨头竞争，国内市场贡献了绝大部分收入。

### 护城河强度 ★★★★

- **技术壁垒**：刻蚀设备研发周期5-7年，中微已攻克5nm及以下先进制程，是国内唯一进入国际一线客户的刻蚀设备商
- **客户粘性**：设备验证周期长，一旦导入产线，替换成本极高
- **专利护城河**：公司积累了大量核心专利，且在设计上刻意区别于国际竞品，避免侵权同时构筑自身壁垒
- **但另一方面**：在全球市场，泛林和东京电子合计占刻蚀市场70%+份额；中微公司的成长高度依赖国产替代政策推动和地缘政治环境

### 管理层质量 ★★★★★

创始人尹志尧博士（60岁回国创业），此前在应用材料担任副总裁13年，是行业顶级技术专家。管理团队杜志游、倪图强等均为国际半导体设备资深人士。公司实行全员持股激励，核心团队稳定性极强。尹志尧在2024年提出"未来5-10年覆盖集成电路关键领域50%-60%设备"的宏大目标，体现了高远的战略视野和执行力。

### 估值吸引力 ★★

PE(TTM)约93倍，即使按2026年一致预期净利润30.75亿元计算，前瞻PE仍有64倍。半导体设备是公认的好赛道，但市场已给出极高估值。对比全球龙头泛林（Lam Research）PE约25倍，中微的估值溢价来自"国产替代"逻辑和更高的增速预期。如果国产替代进程放缓或行业周期下行，估值将面临显著压力。

### 综合评分 ★★★★

### 一句话总结

国产刻蚀设备的绝对龙头，管理层是稀缺的顶级技术创业团队，但当前估值已充分反映国产替代的乐观预期，需要等待更好的价格窗口。

## 3. 澜起科技（688008）— 内存接口芯片

### 基本信息

| 指标 | 数据 |
|------|------|
| 市值 | 约2600亿元（股价约248元） |
| PE(TTM) | 约118倍（基于EPS 2.11元） |
| 股息率 | 0.28% |

### 生意质量 ★★★★★

内存接口芯片是服务器内存条的必备组件，每根DDR5内存条都需要配套1颗RCD芯片和多颗数据缓冲芯片。全球仅澜起和瑞萨两家供应商，形成寡头垄断格局。毛利率高达60%+，净利率35%，接近芯片设计公司的最佳水平。商业模式简单清晰：设计芯片→交给代工厂生产→卖给内存厂和服务器厂商。

这是一个真正的"好生意"：需求随服务器出货量和DDR5渗透率增长而增长，市场集中度极高，产品标准化程度高。2026Q1净利润8.47亿元（同比+61%），全年预期约30亿元。

### 护城河强度 ★★★★★

- **双寡头垄断**：全球仅澜起和瑞萨可提供完整DDR5内存接口芯片方案，壁垒极高
- **标准制定者**：参与JEDEC标准制定，是DDR5标准的重要贡献者
- **认证壁垒**：产品需通过英特尔、AMD平台认证，新进入者面临数年认证周期
- **技术迭代优势**：从DDR4到DDR5的成功切换证明了技术延续性
- **但另一方面**：如果未来DDR6标准变化导致技术路线转换，或者英特尔/AMD自研接口芯片，垄断格局可能被打破

### 管理层质量 ★★★★

创始人杨崇和博士，1997年首次创业成功（被IDT以8500万美元收购），2004年创立澜起。拥有连续创业成功经验和深厚的半导体行业积累。管理哲学清晰："预判市场趋势、组建强大团队、保持资金充裕"。研发人员占比71%，研发投入约15%（高于同行）。

但需注意：杨崇和持美国国籍（中国永久居留权），在当前地缘政治环境下是否存在潜在风险值得关注。

### 估值吸引力 ★★

当前PE(TTM)约118倍，按2026年预期净利润30亿元计算，前瞻PE约87倍。即使按更乐观的2027年预期38亿元，前瞻PE仍有68倍。寡头垄断的好生意值得溢价，但当前估值意味着即使未来两年利润翻倍，PE也还在50倍以上。市场给予了极高的长期成长预期。

### 综合评分 ★★★★

### 一句话总结

全球内存接口芯片双寡头之一，生意质量是六家中最佳（高毛利+寡头垄断+标准制定），但估值高企，适合长期跟踪、在市场恐慌时果断买入。

## 4. 新易盛（300502）— 高速光模块

### 基本信息

| 指标 | 数据 |
|------|------|
| 市值 | 约5000亿元（股价约564元） |
| PE(TTM) | 约40倍（基于2025年利润约94-98亿） |
| 股息率 | 约0.1%（估计） |

### 生意质量 ★★★☆

与中际旭创同一赛道，商业模式类似——为数据中心提供高速光模块。不同之处在于：新易盛聚焦中短距数据中心市场，走LPO（线性驱动光模块）技术路径，强调成本控制和垂直整合（光芯片制造→封装→模块全链条覆盖）。

2025年营收约269亿元，净利润约94-98亿元。2026年机构预期净利润168-178亿元（同比+70-80%），增长极快。毛利率38%，净利率30%+，在光模块行业中表现优异。

但需注意：相比中际旭创的深度客户绑定，新易盛的客户集中度和客户结构信息披露较少，且在1.6T高端市场的话语权可能弱于旭创。

### 护城河强度 ★★★

- **垂直整合**：从光芯片到模块的全产业链覆盖，成本优势明显
- **技术差异化**：LPO路径在功耗和成本上有优势，适合短距应用
- **产能规模**：快速扩产能力强
- **但另一方面**：光模块行业竞争激烈，新易盛在高端市场（1.6T、CPO）的布局不如中际旭创深入；创始人曾因违规减持被处罚，核心技术人员曾有法律纠纷，对长期信任构成隐忧

### 管理层质量 ★★★

董事长高光荣（中专学历，1998年创业，2008年联合创立新易盛）。2023年因违规减持被证监会罚没3149万元（隐瞒家族信托持股+超限减持），这在管理层诚信方面是明显减分项。总经理黄晓雷为技术专家出身（东南大学+电子科大硕士），但早年曾因侵犯商业秘密罪被判刑。

管理层的技术执行力不错（从10G到800G的多代产品迭代表现出色），但诚信记录存在瑕疵。段永平的标准是"不和不诚信的人做生意"，这是一个值得警惕的信号。

### 估值吸引力 ★★★

PE(TTM)约40倍，按2026年预期净利润170亿元计算，前瞻PE约29倍。在光模块三兄弟中估值最低，且增速不弱，看起来具有一定的估值吸引力。但需考虑管理层风险折价和行业周期性。

### 综合评分 ★★★

### 一句话总结

光模块赛道的二号种子，垂直整合带来成本优势，估值相对合理，但管理层诚信瑕疵是硬伤，不符合"买股票就是买人"的核心原则。

## 5. 天孚通信（300394）— 光无源器件

### 基本信息

| 指标 | 数据 |
|------|------|
| 市值 | 约3100亿元（股价约400元） |
| PE(TTM) | 约110-115倍（基于近四季利润约21.7亿） |
| 股息率 | 0.42% |

### 生意质量 ★★★★★

天孚通信的商业模式是光通信的"卖零件"生意：提供超精密光无源器件和光引擎组件，是光模块厂商的上游供应商。毛利率52%、净利率38%——这在硬件制造业中极为罕见，接近芯片设计公司的水平。本质上是靠超精密制造工艺构建壁垒的"隐形冠军"。

2025年营收51.63亿元（同比+59%），净利润20.17亿元（同比+50%）。2026Q1净利润4.92亿元（同比+46%），增速依然强劲。以2025年对外销售收入计，公司以11.7%的市场份额位居全球光器件供应商第一。

客户结构值得关注：2024年仅Fabrinet一家贡献了61.69%的营收，客户集中度极高，既是核心竞争力的证明（全球顶级精密制造商选择天孚），也是风险点。

### 护城河强度 ★★★★

- **超精密制造工艺**：在精密陶瓷、工程塑料、复合金属、光学玻璃等领域积累的核心工艺，对手难以短期复制
- **全球第一市场份额**：光器件市场份额11.7%全球第一
- **客户认证壁垒**：一旦导入顶级客户供应链，替换成本高
- **但另一方面**：客户集中度过高（单一客户占60%+）是明显风险，如果Fabrinet转单或分散供应商，业绩将受重大冲击

### 管理层质量 ★★★★

创始人邹支农（董事长+总工程师）和欧洋（总经理）夫妇联合创业，2005年成立。20年专注光通信精密器件，体现了"专注做好一件事"的经营哲学。公司连续七年荣获"中国光通信最具竞争力企业10强"榜首。管理层稳定、专注、低调，符合价值投资者对优秀管理层的期待。

2026年启动港股上市，显示管理层在资本运作上有积极规划。

### 估值吸引力 ★

PE(TTM)高达110-115倍，处于历史估值94%高分位。即使按2026年预期净利润32-33亿元计算，前瞻PE仍有约95倍。即使按更乐观的2027年预期41亿元，前瞻PE也在75倍。

生意确实好，但当前价格已完全透支了未来两年的增长。市场容错率极低——任何低于预期的季报都可能引发股价大幅调整。

### 综合评分 ★★★☆

### 一句话总结

光通信领域的"精密制造隐形冠军"，生意质量优秀（高毛利+全球第一），但115倍的估值意味着零安全边际，需耐心等待周期回调或业绩大幅释放后的合理价位。

## 6. 工业富联（601138）— AI服务器（豁免C通过）

### 基本信息

| 指标 | 数据 |
|------|------|
| 市值 | 约1.05万亿元（股价约64元） |
| PE(TTM) | 约30倍 |
| 股息率 | 1.93% |

### 生意质量 ★★★

工业富联的商业模式本质是AI服务器的ODM/OEM代工。全球AI服务器市场份额超40%，是英伟达GB系列和微软Azure的核心代工厂。2026Q1营收2511亿元（同比+57%），净利润106亿元（同比+103%），增速惊人。

但需注意关键弱点：毛利率仅7%，净利率仅3.8%（这也是为何去劣筛选需要"豁免C"才通过）。这是典型的"大而不肥"生意——营收万亿级别，但利润率极薄。每一台服务器赚的钱很少，靠巨大的出货量取胜。如果下游需求波动或客户压价，利润弹性有限。

### 护城河强度 ★★★

- **规模壁垒**：年产能超200万台AI服务器，全球无出其右
- **客户深度绑定**：联合研发切入设计初期，前五大客户贡献超60%营收
- **全球化布局**：多国生产基地分散地缘风险
- **良率与交付能力**：大规模量产的良率控制是核心竞争力
- **但另一方面**：代工模式天然缺乏定价权，核心利润由品牌方（英伟达、云厂商）获取；毛利率7%意味着几乎没有议价能力；如果出现竞争对手（如纬创、广达）产能扩张，利润率可能进一步承压

### 管理层质量 ★★★

背靠鸿海/富士康体系，拥有全球制造管理经验。但作为鸿海集团上市平台，独立性存疑——公司治理受母公司影响较大。郭台铭时代的强势管理风格虽然保障了执行力，但也意味着重大决策可能更多服务于集团利益而非A股中小股东利益。

### 估值吸引力 ★★★★

PE(TTM)约30倍，按2026年预期净利润515亿元计算，前瞻PE约21倍。在六家公司中估值最低，且有近2%的股息率。对于一个营收万亿、利润增速45%+的公司来说，21倍前瞻PE看起来合理甚至偏低。

但低估值有低估值的道理：3.8%的净利率意味着代工本质未变，长期ROE天花板有限。

### 综合评分 ★★★

### 一句话总结

全球AI服务器代工的绝对规模王者，估值合理且有股息，但7%毛利率的代工生意本质决定了它是"赚辛苦钱"，不符合"好生意"的最高标准。
